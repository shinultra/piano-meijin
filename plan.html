ピアノゲーム実装計画

1. 目的

ブラウザベースのピアノゲームをHTML/CSS/JavaScriptで実装します。PCではキーボード入力、スマートフォン・タブレットでは画面下部の鍵盤タップで演奏できるようにします。

2. ゲームの流れ

  1. 曲選択画面で楽曲を選択
  2. ゲーム開始後、譜面に合わせてマーカー（ノート）が落下
  3. タイミングよくキーを押すと得点が加算
  4. 選択した曲に合わせたイメージ画像を背景として表示
  5. スコアが100点満点中60点以上なら合格
  6. 1回目は練習ステージ、合格後は次のステージへ
  7. 曲が終わったら、結果画面から次の曲選択画面に戻る

3. 画面構成

曲選択画面

  • 曲タイトル一覧
  • 練習ステージ表示
  • クリアで次のステージをアンロック

ゲーム画面

  • 画面上部：ノート落下エリア
  • 画面下部：鍵盤
  • スコア・判定・合格ラインの表示

背景イメージ

  • 曲ごとにテーマの異なる背景画像を用意
  • ゲーム画面全体に背景を設定して臨場感を演出
  • 曲選択時に各曲のイメージを見せる

結果画面

  • 最終スコア表示
  • 60点以上で合格判定
  • 再挑戦または次のステージへ

4. 操作方法

PC

  • キーボード入力で演奏
  • 例：1 2 3 4 5 6 7 8 を ド レ ミ ファ ソ ラ シ ド に対応

スマホ / タブレット

  • 画面下部の鍵盤をタップ
  • 押した鍵盤に応じて音が再生される

5. 譜面と判定

譜面データ

楽曲ごとにノートのキーと発音タイミングを定義します。

[{ key: 'C', time: 1.25 }, { key: 'D', time: 2.00 }, ...]

ノート落下

  • ノートは上から下へ一定速度で移動
  • 鍵盤位置に到達したタイミングで判定

判定基準

  • Perfect: ±0.1秒
  • Good: ±0.2秒
  • Miss: それ以外

6. スコア設計

  • 100点満点を基準にノート配点を決定
  • 判定ごとに得点を加算
  • 例：Perfectは高得点、Goodは中得点、Missは0点
  • 最終スコアが60点以上で合格

7. 音声再生

  • 鍵盤を押すと対応する音が再生される
  • ブラウザのAudio APIを利用
  • 音声ファイル例：do.mp3, re.mp3, mi.mp3

8. 技術スタック

  • HTML / CSS / JavaScript
  • ファイル構成例：
    • index.html
    • style.css
    • script.js
    • songs.js
    • sounds/

9. 次のステップ

  1. 基本画面レイアウトを作成
  2. 曲選択と譜面データの構造を実装
  3. ノート落下と入力判定の仕組みを実装
  4. 音声再生とスコア計算を追加
  5. モバイル対応のタッチ操作を実装
